# 🌟 Digital Hana 路 3기: 8조 백엔드 실습 게시판 프로젝트 🌟

## 📍 Overview

회원가입/로그인 및 비회원 주문이 가능합니다.

식사 장소를 선택하고, 메뉴 선택하면 주문이 완료됩니다.

주문 금액의 1%은 적립됩니다.

관리자 페이지가 별도로 존재합니다.
